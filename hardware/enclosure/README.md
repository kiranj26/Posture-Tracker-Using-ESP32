# Enclosure

3D printable enclosure files live here. Designed for shoulder clip mounting.

## File naming convention

```
clip_v1.step           ← STEP file (universal, import into any CAD tool)
clip_v1.stl            ← STL for slicing and printing
clip_v1.f3d            ← Fusion 360 source (if designed in Fusion 360)
```

## Target dimensions

```
PCB pocket:     47 × 37 × 9mm  (1mm clearance each side of PCB)
Battery slot:   37 × 27 × 6mm  (behind PCB pocket)
Overall:        45 × 38 × 16mm
Clip opening:   15mm max gap
Material:       PETG
```

## Design requirements checklist

- [ ] PCB slides in without screws (friction fit)
- [ ] USB-C port accessible from bottom edge
- [ ] Both buttons accessible on front face
- [ ] Speaker grille on top face (5-8 holes, 1mm dia)
- [ ] LRA motor recess on back face (contacts body)
- [ ] Spring clip mechanism fits 5-15mm fabric thickness
- [ ] No sharp edges
